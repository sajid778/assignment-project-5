Important FAQ

What is the Document Object Model(DOM)?

The Document Object MOdel is a programming interface for web document. The DOM is a tree like structure. When a webpage loads in our browser it turns the HTML code into a structure it can understant like a tree. WIth the DOM we can use JavaScript to interact with the webpage.

How so you Select an element from the DOM?

We can select an element from the DOM by ID,by Class, by TagName,by querySelector.Select by id we have to write dcoument.getElementById(), and by class name we have to write .getElementByClassName(), by tag name we have to write .getElementByTagName(), by query selector we have to write for single element .querySelector() and for multiple elements.querySelectorAll().

What is event delegation in the context of the DOM, and why is it useful?

event delegation is a smart way to handle events in the DOM. instead of adding to every single element, we can add the event to a parent element and it will handle all the clicks for its child elements. event delegation is vary useful beacuse it is faster and time consuming.by using this it is easier to manage. and we have to write less code for using event delgation.

How do you manipulate an element's attribute and styles using the DOM?

We can use setAttribute(), getAttribute(), removeAttibute() for manipulate an element's attribute like src, href, alt etc.we can manipulate styles by directly modifying element.style.property.
